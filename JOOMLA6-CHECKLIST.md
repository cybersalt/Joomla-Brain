# Joomla 6 Development Checklist

This checklist covers best practices for Joomla 6 extension development and packaging. Update and expand as new Joomla 6 features and requirements are released.

- [ ] Verify minimum PHP version compatibility
- [ ] Use Joomla 6 manifest version
- [ ] Test on clean Joomla 6 install
- [ ] Validate XML manifests
- [ ] Follow new Joomla 6 coding standards
- [ ] Document migration steps from Joomla 5
- [ ] Update build/package scripts for Joomla 6
- [ ] Review new extension points and APIs
- [ ] Add any project-specific requirements
