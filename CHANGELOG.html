v1.2.0 2025-12-18

üöÄ New Features

  • Module Guide - Subform Fields: Added comprehensive documentation for repeatable/sortable subform fields
    • buttons="add,remove,move" for drag-and-drop reordering
    • Processing subform data in Dispatcher with type switching
    • Combining with showon for conditional nested fields
  • Module Guide - Grouped List Fields: Added groupedlist field type documentation
    • Renders proper HTML <optgroup> elements
    • Warning about fancy-select not supporting grouped options properly - Choices.js doesn't handle disabled separator options
  • Module Guide - Conditional Fields (showon): Added complete showon attribute documentation
    • Basic value matching: showon="field:value"
    • Multiple values (OR): showon="field:a,b"
    • Negation: showon="field!:value"
    • AND/OR conditions: [AND] and [OR] operators
  • Module Guide - Live Preview: Added admin form live preview techniques
    • MutationObserver for catching Joomla's dynamic form field updates
    • Click/change/input events for immediate response on form inputs
    • Polling fallback for color pickers and other widgets that don't fire standard events
  • Joomla Skill - Dispatcher Pattern: Updated module section to recommend Dispatcher pattern over Helper pattern
    • Complete service provider example
    • Namespace mapping explanation (Site auto-added by Joomla)
    • Manifest files section for Dispatcher modules

  • Updated JOOMLA5-MODULE-GUIDE.md with learnings from cs-world-clocks module development
  • Updated joomla-development.md skill with subform, groupedlist, showon, and Dispatcher pattern
